## O que é o Fellow Pay?

O **Fellow Pay** é o produto final do **Grupo Fellow** — uma plataforma completa de pagamentos que inclui dashboard administrativo, checkout white-label e gestão financeira para lojistas e marketplaces.

> Enquanto a **Fellow Core** é a API de integração para desenvolvedores, o **Fellow Pay** é a experiência completa para o comerciante.

## Fellow Core vs Fellow Pay

| Aspecto        | Fellow Core                        | Fellow Pay                              |
|----------------|------------------------------------|-----------------------------------------|
| **Público**    | Desenvolvedores                    | Lojistas e gestores                     |
| **Interface**  | API REST                           | Dashboard web + Checkout                |
| **Uso**        | Integração programática            | Operação do dia-a-dia                   |
| **Acesso**     | Via API Key                        | Via login no painel                     |
